## lzr pac

PAC file generation and serving

### Synopsis

Generate or serve a PAC (Proxy Auto-Configuration) file so browsers can auto-select the lazyray proxy. Use the subcommands to print a PAC file or to serve it over HTTP.

### Options

```
  -h, --help   help for pac
```
