## lzr proxy status

Show system proxy status

### Synopsis

Show the current system proxy state: whether the OS HTTP, SOCKS, and PAC proxy settings are enabled and which host and ports they point at. Use it to confirm whether 'lzr proxy on' took effect.

```
lzr proxy status [flags]
```

### Examples

```
  lzr proxy status
```

### Options

```
  -h, --help   help for status
```
