## lzr tunnel close

Close all SSH tunnels

### Synopsis

Close every open SSH tunnel that 'lzr tunnel' started. SSH tunnels persist after the command that opened them exits, so use this to tear them all down.

```
lzr tunnel close [flags]
```

### Examples

```
  lzr tunnel close
```

### Options

```
  -h, --help   help for close
```
