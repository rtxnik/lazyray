## lzr service install

Install xray as a system service

### Synopsis

Install xray-core as a user-scoped system service so it starts automatically on login. The service runs under your user account and does not require root; if installation fails, run 'lzr doctor' and see TROUBLESHOOTING.md.

```
lzr service install [flags]
```

### Examples

```
  lzr service install
```

### Options

```
  -h, --help   help for install
```
