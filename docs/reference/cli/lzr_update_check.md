## lzr update check

Check for xray-core updates

### Synopsis

Compare the installed xray-core against the pinned release for this platform and report whether an update is available. This does not download anything; run 'lzr update apply' to install.

```
lzr update check [flags]
```

### Examples

```
  lzr update check
```

### Options

```
  -h, --help   help for check
```
