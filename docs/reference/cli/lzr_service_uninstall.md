## lzr service uninstall

Remove xray system service

### Synopsis

Remove the user-scoped system service that starts xray-core on login. This stops xray-core from autostarting but does not remove any proxy profiles or settings.

```
lzr service uninstall [flags]
```

### Examples

```
  lzr service uninstall
```

### Options

```
  -h, --help   help for uninstall
```
