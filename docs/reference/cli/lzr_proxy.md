## lzr proxy

System proxy configuration

### Synopsis

Manage the system proxy — the OS-level proxy settings that route applications through lazyray's local ports. Use the subcommands to turn the system proxy on or off and to inspect its current state.

### Options

```
  -h, --help   help for proxy
```
