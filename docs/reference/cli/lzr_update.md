## lzr update

Manage xray-core updates

### Synopsis

Manage the pinned xray-core engine that lazyray runs as its proxy core. Use 'lzr update check' to compare the installed xray-core against the pinned release, and 'lzr update apply' to download and install it.

### Options

```
  -h, --help   help for update
```
