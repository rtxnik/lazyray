## lzr proxy off

Disable system proxy

### Synopsis

Disable the system proxy: remove the OS-level HTTP and SOCKS proxy settings that lazyray applied, restoring direct connections. Run this when you want applications to stop routing through lazyray.

```
lzr proxy off [flags]
```

### Examples

```
  lzr proxy off
```

### Options

```
  -h, --help   help for off
```
