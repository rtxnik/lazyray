## lzr service status

Show service status

### Synopsis

Report whether the user-scoped xray-core autostart service is installed and whether it is currently running. For full diagnostics of the running proxy use 'lzr doctor'.

```
lzr service status [flags]
```

### Examples

```
  lzr service status
```

### Options

```
  -h, --help   help for status
```
