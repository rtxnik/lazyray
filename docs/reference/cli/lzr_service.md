## lzr service

Manage xray system service (autostart)

### Synopsis

Manage the user-scoped system service that starts xray-core automatically on login. Use 'lzr service install' to enable autostart, 'lzr service uninstall' to remove it, and 'lzr service status' to check it.

### Options

```
  -h, --help   help for service
```
