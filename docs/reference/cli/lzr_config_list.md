## lzr config list

List all profiles

### Synopsis

List every proxy profile in the profile store, marking the default with an asterisk. Use it to see what you can switch to or export, and to find the exact profile name other commands expect.

```
lzr config list [flags]
```

### Examples

```
  lzr config list
  lzr config list --json
```

### Options

```
  -h, --help   help for list
      --json   Output in JSON format
```
