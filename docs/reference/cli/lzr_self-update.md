## lzr self-update

Update lazyray to the latest version

### Synopsis

Check for and install the latest version of lazyray from GitHub releases.

```
lzr self-update [flags]
```

### Examples

```
  lzr self-update
```

### Options

```
  -h, --help   help for self-update
```
